TrooWork is a program that has been created to greatly aid Housekeeping and Environmental Services leaders to maximize the productivity of their workforce. TrooWork is a Cloud based web application that provides real time data through barcode scanning, photo uploads, customizable supervisory inspection modules, location monitoring of staff members via GPS, and many other helpful tools. The program has been developed with the end user in mind by considering technologically challenged staff members.

• Simple to operate user interface with front line staff as the number one priority
• Advanced filter operations utilizing multiple criteria for viewing data
• Easy to search daily work order progress reports
• Ensuring work is completed by actually “putting” the employee in the work area through barcode placement
• Locate employees geographically via GPS tracking
• Tracking and scheduling of employee events, trainings, and meetings
• On the fly inspections through customizable templates with real time reports generated and sent to staff members via email
• Multiple application interfaces for manager, supervisor, and staff members

TrooWork has been designed for use with Apple devices such as iPhone, iPad, and iPods. The application is compatible for all above listed devices running iOS 8.0 and above. The program has been created with the end user in mind, attempting to be as simple, efficient, and straight forward as possible.
